# S-DES-Encryption-Decryption
Implementation of the Simplified DES (S-DES) encryption and decryption algorithm with key generation, encryption, and decryption processes.

## Features
- Key Generation
- Encryption Process
- Decryption Process
- Initial and Final Permutations
- Expansion Permutation
- S-Boxes
- XOR Operations

## Technologies Used
- Python

## Algorithm Steps
1. Generate subkeys (K1 and K2)
2. Apply Initial Permutation (IP)
3. Perform fk function
4. Swap left and right parts
5. Apply fk again
6. Apply Final Permutation (IP⁻¹)

## Educational Purpose
This project is created for learning and understanding how block cipher encryption works using the simplified DES algorithm.

## Example
Input Plaintext:
```text
10101010
```

Encrypted Ciphertext:
```text
01110101
```
